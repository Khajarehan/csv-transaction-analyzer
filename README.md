# CSV Transaction Analyzer

A Java-based application that processes financial transactions from a CSV file and generates useful spending insights.

## Features
- Category-wise spending analysis
- Monthly expense aggregation
- Highest spending category detection
- Unusual transaction detection

## Tech Stack
- Java
- HashMap
- File Handling (BufferedReader)

## How It Works
- Reads transaction data from a CSV file
- Converts data into Transaction objects
- Uses HashMap for efficient aggregation and analysis
- Outputs structured financial insights

## How to Run
1. Place your CSV file in the `data/` folder
2. Run `Main.java`
